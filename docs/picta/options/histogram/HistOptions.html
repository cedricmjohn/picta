HistOptions

case class HistOptions(orientation: HistOrientation = VERTICAL, cumulative: Opt[Cumulative] = Blank, histnorm: Opt[HistNormType] = Blank, histfunc: Opt[HistFunction] = Blank, xbins: Opt[Xbins] = Blank, ybins: Opt[Ybins] = Blank) extends Component with Product with Serializable

This class sets the histogram-specific options for a histogram.

orientation

: Specifies whether the histogram is vertical or horizontal.

cumulative

: This takes in a Cumulative type which specifies how the histogram cumulative function works.

histnorm

: Specifies the type of normalization for the histogram series.

histfunc

: Specifies the binning function for the histogram series.

xbins

: An Xbin specified by the user.

ybins

: A Ybin specified by the user.
