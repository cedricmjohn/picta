Chart

final case class Chart(data: List[Series] = Nil, layout: Layout = Layout(), config: Config = Config(), animated: Boolean = false, transition_duration: Int = 100) extends ChartInterface with Product with Serializable

This is the chart base class with which we can compose all other elements of the grammar that determine how the chart is displayed.

data

: This is a list of 'series' type that we wish to add to the plot.

layout

: This is the layout that determines how the chart is visually laid out.

config

: This is the config that controls how the plot behaves with user interaction.

animated

: Specifies whether the chart is animated or static.

transition_duration

: If the chart is animated, this specifies the time (ms) it takes to change frame.
